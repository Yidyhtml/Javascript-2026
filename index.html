Javascript

Javascript es un lenguaje de programación dinámico y de alto nivel para crear interfaces interactivas y dinámicas.

Es uno de los tres pilares fundamentales del desarrollo web.

  • 1 HTML: Define la estructura del contenido. Son los elementos que aparecen en la página web.
  • 2 CSS: Define la apariencia, colores, tamaños, estilos, etc.
  • 3 JS: Define el comportamiento, es decir, lo que hace la página o sus elementos.

Características de Javascript

  1. (☞ﾟヮﾟ)☞Lenguaje interpretado: No se necesita compilar. El navegador lo entiende directamente.
  2. (☞ﾟヮﾟ)☞Basado en objetos: Maneja los elementos de html como objtos: window, date, array, class.
  3. (☞ﾟヮﾟ)☞Dinámico: Podemos modificar el contenido en tiempo real. (sin recargar)
  4. (☞ﾟヮﾟ)☞Versatil y fácil de aprender: Aunque nacio especialmente para la web, se usa también para servdores, aplicaciones móviles, videojuegos, animación y para IA.
